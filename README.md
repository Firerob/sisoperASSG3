Para compilar el simulador de planificación de disco basta con ejecutar en la consola el comando g++ disk.cpp, lo cual generará el archivo ejecutable del programa. Para ponerlo en funcionamiento, simplemente se debe ejecutar a.exe seguido de la posición inicial del cabezal del disco; por ejemplo, a.exe 1500 iniciará la simulación con el cabezal ubicado en el cilindro 1500. Al hacerlo, el programa generará automáticamente una secuencia aleatoria de 1000 solicitudes y las procesará utilizando los algoritmos FCFS, SCAN y C-SCAN. Posteriormente, mostrará en pantalla el movimiento total del cabezal realizado por cada algoritmo y, adicionalmente, generará imágenes en formato PNG que ilustran el comportamiento del movimiento del cabezal y comparan el rendimiento de los tres algoritmos. Este proceso permite visualizar de forma clara el impacto de cada técnica de planificación sobre la eficiencia del disco.
